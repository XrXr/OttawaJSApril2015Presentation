ES6 features I would use everyday

This is an expansion of a section of a talk I did while back. I got exposed to some of the ES6 features while making Firefox add-ons. If you look at the JavaScript in Firefox internals, they use a lot of the features I'm going to be talking about today. Some of these features have been inside Firefox for quite a while, just not for webpages

All of the features I talk about in this talk are usable right now

In the latest browsers and latest Node
